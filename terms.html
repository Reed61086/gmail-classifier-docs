2. Service Description

The Gmail Email Classifier is a Chrome extension that:

  • Helps organize your Gmail inbox through AI-powered classification
  • Creates and manages labels in your Gmail account
  • Processes emails locally on your device

3. Gmail API Usage

The extension uses the Gmail API to:

  • Read email content for classification
  • Create and apply labels
  • Modify email organization

All processing is done locally on your device, and no email content is stored or transmitted to external servers.
